Project Description:
This project focuses on analyzing an e-commerce dataset to understand customer purchasing behavior, sales patterns, and order trends. The analysis includes data cleaning, preprocessing, and exploratory data analysis (EDA) to identify meaningful insights. Various factors such as products, payment methods, order status, coupon usage, and total sales were examined to support better business decisions.
Objectives:
Clean and preprocess the dataset
Analyze customer purchase behavior
Study product sales performance
Examine payment methods and order status trends
Identify the impact of coupons and referral sources
Generate insights using visualizations and dashboards
Tools Used:
SQL / Power BI / Python (keep only the tools you actually used)
Microsoft Excel
Dataset Features:
Order ID
Customer ID
Product details
Quantity and pricing
Payment method
Order status
Coupon code usage
Referral source
Total price
